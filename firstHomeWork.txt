
1. Прежде чем скопировать на свой компьютер репозиторий из GitHub необходимо убедиться что git установлен на компьютере и в GitHub  сгенерирован ssh key . Затем необходимо получить SSH из репозитория и в терминале перейти в директорию где будет находиться сам clone  и выполнить команду git clone +  SSH  репозитория.

2.Чтоб создать git  репозитория из локального каталога  необходимо :
		
		a)  в терминале перейти в директорию где находиться отправляемый проект
		b)  Выполнить команду git init 
		c)  Затем заполнить репозиторий всеми файлами которые необходимо добавить git add .
		d)  Сделать git commit -m "" 
		e)  git remote add origin git@github.com: + путь к репозиторию 
		f)  Git push -u origin main (если загружаем первый раз )
			Все следующие добавления необходимо использовать только git push

3. Чтоб получить изменения с GitHub  необходимо вести git pull

4. git status : показывает что происходит с файлами в текущей директории 
   
   git log  : показывает список всех изменений 

   git remote -m : Показывает по какому адресу хранится текущий проект
